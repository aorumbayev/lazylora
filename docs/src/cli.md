# CLI Commands

LazyLora provides several command-line options:

## Basic Usage

```bash
# Run with default settings
lazylora

# Show help
lazylora --help

# Show version
lazylora --version
```

## Update Commands

```bash
# Check for updates
lazylora update

# Update to the latest version
lazylora update --install
```

## Options

| Option | Description |
|--------|-------------|
| `--help` | Display help information |
| `--version` | Display version information |
