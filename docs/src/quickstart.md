# Quick Start

After installation, you can start LazyLora by running:

```bash
lazylora
```

This will launch the TUI interface connected to Algorand MainNet by default.

## First Steps

1. **Navigate blocks and transactions**: Use arrow keys to move through the list
2. **Switch tabs**: Press `Tab` to switch between blocks and transactions views
3. **View details**: Press `Enter` to see detailed information about a selected item
4. **Search**: Press `f` to search for specific transactions
5. **Switch networks**: Press `n` to change between MainNet, TestNet, and LocalNet

## Live Updates

LazyLora automatically fetches new blocks and transactions in real-time. You can toggle live updates on/off by pressing `Space`.
